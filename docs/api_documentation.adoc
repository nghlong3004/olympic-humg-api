=== Character Encoding

* All request/response use `UTF-8`.

'''

=== Common Data Types

[cols="2,3,5",options="header"]
|===
| Kiểu
| Ví dụ
| Ghi chú

| `string`
| `"Long Nguyen"`
| Chuỗi ký tự UTF-8

| `email`
| `"student01@example.com"`
| Chuỗi, format email

| `date`
| `"2004-03-30"`
| ISO-8601, `YYYY-MM-DD`

| `timestamp`
| `"2025-02-10T12:30:00"`
| ISO-8601, `YYYY-MM-DDThh:mm:ss`

| `enum`
| `"STUDENT"`
| Một trong những giá trị cho phép
|===

'''

=== ErrorResponse Data Type

Tất cả lỗi trong hệ thống trả về theo cấu trúc JSON sau:

[source,json]
----
{
  "timestamp": "2025-02-10T10:05:00Z",
  "status": 400,
  "message": "Email is already registered."
}
----

[cols="2,2,6",options="header"]
|===
| Trường | Kiểu | Ghi chú

| `timestamp`
| datetime
| Thời điểm xảy ra lỗi (ISO-8601)

| `status`
| number
| Mã lỗi HTTP

| `message`
| string
| Thông báo lỗi chung
|===
