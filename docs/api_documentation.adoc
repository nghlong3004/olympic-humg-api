== General Format

=== Base URL

[source]
----
/api
----

Ví dụ đầy đủ cho môi trường dev:

[source]
----
http://localhost:8080/api
----

=== Character Encoding

* Tất cả request/response dùng `UTF-8`.

=== Common Data Types

[cols="2,3,5",options="header"]
|===
| Kiểu
| Ví dụ
| Ghi chú

| `string`
| `"Long Nguyen"`
| Chuỗi ký tự UTF-8

| `email`
| `"student01@example.com"`
| Chuỗi, format email

| `date`
| `"2004-03-30"`
| ISO-8601, `YYYY-MM-DD`

| `datetime`
| `"2025-02-10T12:30:00"`
| ISO-8601, `YYYY-MM-DDThh:mm:ss`

| `enum`
| `"STUDENT"`
| Một trong những giá trị cho phép
|===
